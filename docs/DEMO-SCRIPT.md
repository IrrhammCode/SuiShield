# 🎬 SuiShield — 90-Second Demo Script

- [ ] App running locally (`npm run dev`)
- [ ] Browser in dark mode, clean profile (no bookmarks bar)
- [ ] Window size: 1920x1080 (fullscreen)
- [ ] Close all other tabs
- [ ] Have a Sui address ready: `0x742d35Cc6634C0532925a3b844Bc9e7595f2bC56` (or any real Sui address)
- [ ] Test everything once before recording

---

## 🎬 SCENE 1 — Hook (0:00 - 0:10)

**Screen:** Homepage (`/`)

**Script:**
> "Every day, thousands of people lose funds to scam wallets on Sui. What if you could check any address before you interact — and get a trust score in seconds? Meet SuiShield."

**Action:** Show the homepage, let it breathe for 2 seconds.

---

## 🎬 SCENE 2 — Trust Analysis (0:10 - 0:35)

**Screen:** Navigate to `/analyze`

**Script:**
> "Paste any Sui address. SuiShield's AI agent instantly pulls real-time data from Tatum RPC — balance, transactions, owned objects. It analyzes six risk signals and gives you a clear verdict."

**Action:**
1. Click on the address input field
2. Paste: `0x742d35Cc6634C0532925a3b844Bc9e7595f2bC56`
3. Click "Analyze"
4. Show the loading animation (2-3 seconds)
5. Show the result: trust score card, risk signals, verdict

**Script (continued):**
> "Safe. Score 18 out of 100. Every signal is transparent. And the proof? Stored permanently on Walrus."

**Action:** Scroll down to show risk signals and on-chain proof section.

## 🎞️ Screen Recording Order

Record in this order (easier to edit):

| # | Scene | Duration | What to Record |

| 2 | Analyze | 25s | Paste address → Analyze → Show result |
| 3 | Trust Graph | 15s | Enter address → Trace → Show graph |

**Total: ~90 seconds**

---

## 🎙️ Voiceover Tips

- **Pace:** Not too fast. Pause between scenes.
- **Tone:** Professional but approachable. Think product launch video.
- **Emphasis words:** "any address", "in seconds", "real-time", "11 terabytes", "Tatum", "Walrus"
- **No um/uh:** Practice 2-3 times before recording voiceover

---

## 📹 Recording Tools

- **Mac:** QuickTime (Screen Recording)
- **Windows:** Xbox Game Bar (Win + G) or OBS
- **Browser:** Use Chrome at 100% zoom

---

## ✂️ Editing Tips

- Add subtle zoom-ins on key moments (trust score, graph, activity)
- Add text overlays for key numbers: "6 Signals", "11TB Data", "29 Tatum Integrations"
- Background music: Lo-fi or tech ambient (low volume)
- Transitions: Simple cuts, no fancy effects
